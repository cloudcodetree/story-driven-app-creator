# [EPIC-02] Morse Transmission

## Description
Send messages via multiple output modalities — flashlight, screen flash, audio tones, and
vibration. Users type or speak a message and the app encodes and transmits it as morse code.

## Goal
Enable users to transmit messages in morse code through at least one modality without
knowing morse code — the app handles all encoding.
