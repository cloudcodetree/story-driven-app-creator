# [EPIC-05] Learning & Practice

## Description
Help users build morse code proficiency over time through guided practice,
real-time feedback, and progress tracking.

## Goal
Users who are "willing to learn" can build real morse fluency through the app,
increasing both their engagement and their ability to communicate without the app's help.
