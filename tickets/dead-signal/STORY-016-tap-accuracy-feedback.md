# [STORY-016] Real-Time Feedback on Tap Accuracy

## User Story
As a user practicing morse code,
I want to see real-time feedback on whether my taps are accurate — correct timing, clean dit/dah separation,
so that I can improve my technique.

## Acceptance Criteria
- [ ] Given I'm tapping, when I complete a letter, then I see if it was correct or what I actually tapped
- [ ] Given my timing is inconsistent, when I tap, then a visual indicator shows how close I am to clean timing ratios
- [ ] Given I'm improving, when my accuracy increases, then the app acknowledges the improvement

## Technical Notes
- Compare user's dit:dah ratio against ideal 1:3 ratio
- Show timing consistency score
- Reuse the adaptive threshold visualization from the prototype
