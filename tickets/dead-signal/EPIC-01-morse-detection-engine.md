# [EPIC-01] Morse Detection Engine

## Description
The core signal processing system that decodes incoming morse code from multiple input sources:
tap (touch), audio (microphone), and visual (camera). This is the hardest technical challenge
and the foundation the entire product depends on.

## Goal
Reliably decode morse code from at least one input modality with adaptive tempo handling,
proving the core product is technically viable.

## Stories
- [x] [STORY-001] Tap input detection with adaptive timing
- [ ] [STORY-002] Audio morse detection via microphone
- [ ] [STORY-003] Visual morse detection via camera
- [ ] [SPIKE-001] Test detection accuracy in noisy environments

## Priority: Critical
