# [STORY-018] Onboarding Flow

## User Story
As a new user opening Dead Signal for the first time,
I want a quick onboarding that shows me what this app does and gets me set up,
so that I'm ready to use it before I actually need it.

## Acceptance Criteria
- [ ] Given I open the app for the first time, when the onboarding starts, then I see 3-4 screens explaining the concept
- [ ] Given the onboarding explains the app, when I reach the end, then I'm prompted to create my first channel or try the SOS button
- [ ] Given I want to skip, when I tap "skip," then I go straight to the main screen
- [ ] Given I've completed onboarding, when I reopen the app, then onboarding doesn't show again

## Technical Notes
- Emphasize: "set this up now, use it when you need it"
- Show the SOS button prominently — it's the zero-setup value
- Encourage creating a channel with one contact as the first action
- Store onboarding completion in localStorage
