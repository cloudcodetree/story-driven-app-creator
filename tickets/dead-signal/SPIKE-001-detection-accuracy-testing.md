# [SPIKE-001] Test Detection Accuracy in Noisy Environments

## Question
How reliably can we detect morse code via audio and visual input in real-world conditions —
noisy rooms, outdoor environments, varying distances, multiple light sources?

## Timebox
3 days

## Expected Output
- A test report with accuracy percentages across conditions:
  - Audio: quiet room, moderate noise, loud noise, outdoors
  - Visual: dark room, indoor lighting, outdoor daylight, varying distance (1m, 5m, 10m, 25m)
- A decision on minimum viable conditions for each modality
- Recommendations for signal processing improvements if accuracy is below 90%
