# [EPIC-04] Emergency Mode

## Description
Zero-setup emergency communication for people who don't know morse code.
One-tap SOS, common emergency phrase presets, and automatic incoming SOS detection.
Works without channels — uses universally recognized morse patterns.

## Goal
Any user can send an SOS or emergency message within seconds of opening the app,
with zero prior setup or morse knowledge.
